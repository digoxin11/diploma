История пробок

Описание дипломной работы

Этот сайт визуализирует данные о пробках в городе Алматы на основе данных, собранных для дипломного проекта Кайролдиной Айым.

Используется автоматизированный сбор трафика с онлайн-карт, хранение в Google Sheets и отображение с помощью OpenStreetMap и Chart.js.
